Automatización de pedidos por WhatsApp con IA

Backend que elimina la gestión manual de pedidos para negocios que reciben órdenes por WhatsApp. La IA interpreta los mensajes, registra los pedidos y envía confirmaciones sin intervención humana.

  • Python 3.11
  • FastAPI
  • PostgreSQL
  • OpenAI GPT-4o
  • Meta Cloud API
  • Docker
  • Railway
  • IA que interpreta mensajes de clientes y extrae pedidos automáticamente
  • Máquina de estados: recibido → confirmado → preparación → despachado → entregado
  • API de administración con autenticación JWT, filtros y estadísticas de dashboard
  • Webhooks validados con HMAC-SHA256 y cobertura de pruebas end-to-end

Proyecto desplegado en Railway con Docker. Integra IA generativa en un flujo de negocio real y automatizable.
